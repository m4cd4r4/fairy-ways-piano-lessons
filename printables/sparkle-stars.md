## How to Earn Stars

| Star Type | How to Earn |
|-----------|-------------|
| Weekly Star | Complete all lessons and challenge for that week |
| Courage Star | Try something difficult without giving up |
| Kindness Star | Play gently for Roxy-Rose or others |
| Practice Star | Practice every day for a full week |
| Performance Star | Play for family, friends, or stuffed animals |
| Memory Star | Play a complete song from memory |
| Teacher Star | Teach someone else about the piano |
